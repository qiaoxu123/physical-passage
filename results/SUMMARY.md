## 核心结论

1. **"执行前判断能不能过"可以被模仿学习教会**:S1/S2 认输均 10/10、零误报,零样本则为 0。
2. **语言先验消除颜色捷径**:颜色交换 CNN 0.50 → LoRA 0.975。VLM 靠"物体性"识别而非颜色锚定。
3. **精确空间占据推理谁都没学会**:临界孔两者 ≈0.55;差 1.5cm 的骗人孔照样硬闯。
4. **单视角感知墙与模型大小无关**:专用回归网从固定斜视角 224² 图最多解码 dx 6cm / dz 5cm /
   转角 10°(已收敛),而离散对齐需要 2.5cm / 2.5°;BC 与 LoRA 的旋转关成绩(1/10、0/10)一致卡死。
   DAgger 一轮、本体感知、128→224 分辨率均无效。
